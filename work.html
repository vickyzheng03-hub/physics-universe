Work

In physics, work is done when a force moves an object through a distance.

Key idea

Not every force does work. Work only happens if the object actually moves in the direction of the force.

  • If you push a wall and it does not move, no work is done on the wall.
  • If you push a box and it moves, work is done.
Force
Distance
Work done

Everyday example

Carrying a school bag while walking involves force. Lifting the bag onto a desk does work because the bag moves upward.

Worked example

Question: A force of 10 N pushes a box 3 m. How much work is done?

Step 1: Use W = F × d.

Step 2: W = 10 × 3.

Step 3: W = 30 J.

Main formula
W = F × d
  • W = work (J)
  • F = force (N)
  • d = distance (m)

Mini quiz

If a force does not move the object, is work done?

Answer: No. In physics, work needs movement.
